# CodeVector Product API

## Features
- PostgreSQL Database
- Express.js Backend
- Cursor Pagination
- Category Filtering
- Limit Support

## API Endpoints

GET /products
GET /products?limit=20
GET /products?category=Electronics
GET /products?limit=20&cursor=41776

## Tech Stack
- Node.js
- Express.js
- PostgreSQL

## Run Locally

npm install
npm start
